# react-native-cellrebelsdk

ReactNative wrapper for CellRebelSDK

## Installation

```sh
npm install react-native-cellrebelsdk
```

## Usage

```js
import Cellrebelsdk from "react-native-cellrebelsdk";

// ...

const result = await Cellrebelsdk.multiply(3, 7);
```
